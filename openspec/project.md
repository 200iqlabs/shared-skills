# Shared Skills

## Description
Modular AI agent library for business advisory. Built on Agent Skills standard (agentskills.io).

## Tech Stack
- Agent Skills format (SKILL.md + references/ + scripts/)
- Claude Code CLI
- Bash + Python scripts
- Git submodules for distribution

## Key Decisions
- Skills over MCP servers (lighter context, more control)
- Progressive disclosure (metadata → body → references)
- Descriptions optimized with skill-creator evals
